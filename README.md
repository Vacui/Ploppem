**Code repository of Ploppem, a minimal endless-runner I developed to improve my skills.**

### Game Features
* One touch input endless runner with increasing difficulty
* Game statistics
* Minimal graphics
* Customizable bacgkround, enemis and UI palette
* Multiple languages
